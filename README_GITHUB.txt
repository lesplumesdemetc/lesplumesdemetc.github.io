COMMENT METTRE LE SITE SUR GITHUB
---------------------------------
1. Décompresser ce ZIP sur le PC.
2. Aller sur le dépôt GitHub : lesplumesdemetc.github.io
3. Cliquer sur "Add file" puis "Upload files".
4. Envoyer tout le contenu décompressé.
5. Cliquer sur "Commit changes".
6. Attendre quelques minutes.
7. Vérifier le site : https://lesplumesdemetc.github.io

IMPORTANT
---------
Il ne faut pas envoyer le ZIP directement dans GitHub.
Il faut envoyer les fichiers décompressés.

Le fichier index.html doit être visible directement à la racine du dépôt.
Il ne doit pas être caché dans un sous-dossier.

MODIFIER LES DISPONIBILITÉS
---------------------------
Le fichier à modifier régulièrement est :

disponibilites.js

Exemple :
statut: "Disponible"
statut: "Sur réservation"
statut: "À confirmer"
statut: "Indisponible"

Tu peux aussi modifier :
derniereMiseAJour: "29/05/2026"
