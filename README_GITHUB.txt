SITE : Les Plumes de M & C
Adresse prévue : https://lesplumesdemetc.github.io

CONTENU DU DOSSIER
- index.html : page principale du site
- style.css : design du site
- script.js : fonctionnement du tableau des disponibilités
- disponibilites.js : fichier à modifier régulièrement pour changer les prix/statuts/notes
- assets/img/ : toutes les images utilisées par le site
- .nojekyll : fichier utile pour GitHub Pages

MISE EN LIGNE SUR GITHUB
1. Décompresser ce fichier ZIP sur le PC.
2. Ouvrir le dossier extrait.
3. Envoyer sur GitHub les fichiers visibles dans ce dossier : index.html, style.css, script.js, disponibilites.js, .nojekyll et le dossier assets.
4. Ne pas envoyer uniquement le ZIP sur GitHub : il faut envoyer les fichiers décompressés.
5. Le fichier index.html doit être à la racine du dépôt, pas caché dans un sous-dossier.

MODIFIER LES DISPONIBILITÉS
1. Ouvrir le fichier disponibilites.js.
2. Modifier uniquement les textes entre guillemets.
3. Exemples de statut : Disponible / Sur réservation / Bientôt disponible / Indisponible.
4. Enregistrer puis renvoyer le fichier modifié sur GitHub.

COMPTEUR DE VISITES
Le site contient déjà le script GoatCounter dans index.html.
Pour qu’il fonctionne réellement :
1. Créer un compte sur https://goatcounter.com
2. Choisir le code : lesplumesdemetc
3. Une fois créé, les statistiques seront consultables ici : https://lesplumesdemetc.goatcounter.com

Si tu choisis un autre code GoatCounter, il faudra remplacer dans index.html :
https://lesplumesdemetc.goatcounter.com/count
par :
https://TONCODE.goatcounter.com/count

IMAGES
Les images sont dans assets/img/.
Attention : certaines images portent un filigrane “leboncoin”. Si ces images ne sont pas les tiennes ou si tu veux un rendu plus professionnel, remplace-les par tes propres photos sans filigrane.
